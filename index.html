Vite + React Project

ถ้าเห็นหน้านี้ แปลว่ายังไม่ได้รันผ่าน Vite

โปรเจกต์นี้เป็น React/Vite จึงไม่ควรเปิดด้วย Live Server ที่ลิงก์แบบ 127.0.0.1:5500/index.html เพราะ browser จะอ่านไฟล์ JSX ตรงๆ ไม่ได้

  • ให้ติดตั้ง Node.js ก่อน
  • จากนั้นรัน npm install
  • แล้วรัน npm run dev
  • เปิดที่ http://localhost:5173
  • ถ้าจะ deploy Vercel/Firebase ให้ build จากโปรเจกต์นี้ตาม README
